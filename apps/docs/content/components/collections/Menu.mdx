<Example src="menu/docs/Sections" />

### Dividers

A menu's items can be separated by dividers.

<Example src="menu/docs/Dividers" />

### Item icons

<Example src="menu/docs/ItemIcon" />

### Item end icon

A select item can have non standard end icons can be provided to handle special cases like displaying a list of icons.

### Disabled item

A menu item can be disabled.

<Example src="menu/docs/DisabledItem" />

<Example src="menu/docs/DisabledKeys" />

### Validation

A menu item can be set as invalid using the `isInvalid` prop.

<Example src="menu/docs/Validation" />

### Selection

A menu supports multiple selection modes.
By default, selection is disabled, however this can be changed by settings the selectionMode property to "single" or "multiple".

<Callout variant="information">
When using selection with a `<Menu>` component wraped inside a `<MenuTrigger>`, you must handled the selected keys in controlled mode with the selectedKeys property otherwise the selected keys {`won't`} be persisted through openings.
</Callout>

<Example src="menu/docs/Selection" />

### Section-level selection

Each section in a menu may have independent selection states. This works the same way as [described above](/components/Menu#usage-selection) for the entire menu, but operates at the section level instead.

<Example src="menu/docs/SectionLevelSelection" />

### Custom Trigger

MenuTrigger works out of the box with any pressable component (e.g. Button, Link, etc.).

Custom trigger elements such as third party components and other DOM elements are also supported by wrapping them with the `<Pressable>` component.

<Example src="menu/docs/CustomTrigger" />

### Dynamic items

A menu items can be rendered dynamically.

<Example src="menu/docs/DynamicItems" />

### Controlled

The `selectedKey` and `open` state can be handled in controlled mode.

<Example src="menu/docs/Controlled" />

### Opening a modal

A ModalTrigger component won't work with a menu item. Still, it can be done with a custom modal trigger.

<Example src="menu/docs/OpeningModal" />

### Submenu

A menu can have submenus, which are menus that open when hovering or clicking on a menu item.
