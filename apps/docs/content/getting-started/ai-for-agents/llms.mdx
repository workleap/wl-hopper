We support [LLMs.txt](https://llmstxt.org/) files for making the Hopper documentation available to large language models.

## Directory Overview

The following files are available:

- [llms.txt](/llms.txt): The main LLMs.txt file
- [llms-full.txt](/llms-full.txt): The complete documentation for Hopper

<br/>

Separate docs are available if you have a limited context window:
- [llms-components.txt](/components/index.txt): All React components in Hopper
- [llms-styled-system.txt](/styled-system/index.txt): Styled system, design tokens, responsive styles, and theme customization
- [llms-tokens.txt](/tokens/index.txt): Comprehensive reference for semantic colors, typography, spacing, and other design tokens
